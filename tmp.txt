


The bug you've uncovered is a classic memory-corruption scenario in graph execution frameworks related to partial executions / non-contiguous `TensorView`s!

### The Problem
During full planning, the `Planner` selects CPU kernels like `Add_3D_1D_Inplace`, `Dot_F32_3D`, etc. The `match` functions for these kernels explicitly require the inputs to be strictly contiguous in memory (i.e. `view.isContiguous() == true`). 

When the graph executes *partially* inside the `Session::ensureCacheCoverage` logic (for calculating dirty regions, e.g., the 1 new token added out of a maximum sequence length of 128), it generates a *sliced* `TensorView`. The framework appropriately adjusts the `shape` and `baseOffset` for the view, but the `strides` remain the original array strides. This means the sliced `TensorView` is correctly recognized as non-contiguous!

The bug hits when `findMatchingKernels` attempts to find a kernel to execute the non-contiguous slice. Because the reference ops were written strictly for contiguous layouts, no kernel is matched. `Session::ensureCacheCoverage` then **falls back** to the original kernel chosen by the planner (`inst.kernelId`). This contiguous-only kernel is forced to process the non-contiguous array, leading to complete corruption of memory bounds and overwriting elements across batch/head indices.

### The Solution
We must equip the `TensorView` structures and the C++ CPU/CUDA kernels to handle physical strides seamlessly. 

The fix implements an inline `getStridedIndex` function that correctly translates logical flat indices (from `0` to `totalElements`) back to the physical multidimensional strided index. We then strip the `!inputs[0].view.isContiguous()` assertions from the CPU match implementations and inject correct non-contiguous iteration directly into the execution bodies.

Here are the fixes covering the primary core files and kernel implementations:

```cpp
// File: tensor_graphs_cpp/core/types.hpp
// Insert this right before `countElements` or anywhere near the top of the file

inline uint64_t getStridedIndex(uint64_t flatIndex, const std::vector<uint32_t>& shape, const std::vector<int64_t>& strides) {
    uint64_t stridedIndex = 0;
    uint64_t temp = flatIndex;
    for (int i = static_cast<int>(shape.size()) - 1; i >= 0; --i) {
        if (shape[i] > 1) {
            uint64_t coord = temp % shape[i];
            stridedIndex += coord * strides[i];
            temp /= shape[i];
        }
    }
    return stridedIndex;
}
```

// File: tensor_graphs_cpp/kernels/cpu/reference/add/F32_ND.hpp
#pragma once
#include "core/types.hpp"
#include "core/kernels.hpp"

inline bool matchAddF32_ND(const std::vector<TensorNode> &inputs, const TensorNode &output, const std::unordered_map<uint32_t, uint32_t> &refCounts)
{
    if (inputs.size() != 2) return false;
    if (inputs[0].dtype != DType::FLOAT32 || inputs[1].dtype != DType::FLOAT32 || output.dtype != DType::FLOAT32) return false;
    if (inputs[0].shape != inputs[1].shape || inputs[0].shape != output.shape) return false;
    return true;
}

inline void runAddF32_ND(const std::vector<const void *> &inputs, const std::vector<void *> &outputs,
                         const std::vector<TensorView> &inViews, const std::vector<TensorView> &outViews)
{
    const float *a = static_cast<const float *>(inputs[0]);
    const float *b = static_cast<const float *>(inputs[1]);
    float *out = static_cast<float *>(outputs[0]);
    uint64_t numElements = countElements(inViews[0].shape);

    if (inViews[0].isContiguous() && inViews[1].isContiguous() && outViews[0].isContiguous()) {
        for (uint64_t i = 0; i < numElements; ++i) out[i] = a[i] + b[i];
    } else {
        for (uint64_t i = 0; i < numElements; ++i) {
            out[getStridedIndex(i, outViews[0].shape, outViews[0].strides)] = 
                a[getStridedIndex(i, inViews[0].shape, inViews[0].strides)] + 
                b[getStridedIndex(i, inViews[1].shape, inViews[1].strides)];
        }
    }
}
REGISTER_REF_KERNEL(OpType::ADD, Backend::CPU, matchAddF32_ND, runAddF32_ND);

#pragma once
#include "core/types.hpp"
#include "core/kernels.hpp"

inline bool matchDotF32_3D(const std::vector<TensorNode> &inputs, const TensorNode &output, const std::unordered_map<uint32_t, uint32_t> &refCounts)
{
    if (inputs.size() != 2) return false;
    if (inputs[0].dtype != DType::FLOAT32 || inputs[1].dtype != DType::FLOAT32 || output.dtype != DType::FLOAT32) return false;
    const auto &s0 = inputs[0].shape;
    const auto &s1 = inputs[1].shape;
    const auto &so = output.shape;
    if (s0.size() != 3 || s1.size() != 3 || so.size() != 3) return false;
    if (s0[0] != s1[0] || s0[2] != s1[1]) return false;
    if (so[0] != s0[0] || so[1] != s0[1] || so[2] != s1[2]) return false;
    return true;
}

inline void runDotF32_3D(const std::vector<const void *> &inputs, const std::vector<void *> &outputs,
                         const std::vector<TensorView> &inViews, const std::vector<TensorView> &outViews)
{
    const float *A = static_cast<const float *>(inputs[0]);
    const float *B = static_cast<const float *>(inputs[1]);
    float *Out = static_cast<float *>(outputs[0]);

    uint32_t B_count = outViews[0].shape[0];
    uint32_t M = outViews[0].shape[1];
    uint32_t K = inViews[0].shape[2];
    uint32_t N = outViews[0].shape[2];

    const auto& sA = inViews[0].strides;
    const auto& sB = inViews[1].strides;
    const auto& sOut = outViews[0].strides;

    for (uint32_t b = 0; b < B_count; ++b) {
        for (uint32_t m = 0; m < M; ++m) {
            for (uint32_t n = 0; n < N; ++n) {
                float sum = 0.0f;
                for (uint32_t k = 0; k < K; ++k) {
                    sum += A[b * sA[0] + m * sA[1] + k * sA[2]] * 
                           B[b * sB[0] + k * sB[1] + n * sB[2]];
                }
                Out[b * sOut[0] + m * sOut[1] + n * sOut[2]] = sum;
            }
        }
    }
}
